====== Типовое оборудование ======

Здесь приведен перечень типового оборудования и способ подключения

===== Подключение сухих контактов =====

К этой группе датчиков относятся:    

| **Тип**                | **Пример**                      | **Назначение** |
| Герконы                | ![](:photo:reed_1.jpg)          | используется как средство обнаружения магнитного поля, успешно применяется для задач расстановки предметов, снабженных небольшими магнитами |
| Концевые переключатели | ![](:photo:limit_switch_1.jpg)  | используется для замыкания контакта механическим образом, прямым контактом другого предмета, например, закрытие двери, опускание рычага и т.д. |
| Тумблеры               | ![](:photo:tumbler_1.jpg)       | используется для возможности игрокам специально воздействовать на процесс, тумблер имеет два стабильных положения вкл/выкл |
| Кнопки                 | ![](:photo:button_1.jpg)        | используется для возможности игрокам специально воздействовать на процесс, кнопка может иметь одно из двух состояний нажата/отжата |
| Галетные переключатели | ![](:photo:rotary_switch_1.jpg) | используется для возможности игрокам специально воздействовать на процесс, галетный переключатель может иметь несколько состояний |

... и прочие датчики, которые просто замыкают цепь. При подключении сухих контактов используется элементарная схема:  
![](:layout:electro_drycontact.png) 

**VREF** - это любое опорное напряжение, которое будет скоммутировано датчиком на **SIGNAL** в замкнутом состоянии. Именно **SIGNAL** как правило заводится и в контроллер управления для съема информации. Сухие контакты являются наипростейшим вариантом реализации триггеров в квестах.

Допускается последовательное и параллельное подключение сухих контактов для реализации различной логики. Схематически это можно представить следующим образом:  
![](:layout:electro_drycontact_multi.png)  

для того, чтобы триггер сработал в вышеприведенной схеме необходимо замкнуть 1+2 или 3+4. Например, если 1 и 2 - это галетные переключатели, 3 - геркон, а 4 - концевой переключатель, то решением будет являться одно из двух условий: либо выставлены галетные переключатели 1 и 2 в нужные положения, либо при условии размещение магнита около геркона 3 и нажатии на концевой выключатель 4.
