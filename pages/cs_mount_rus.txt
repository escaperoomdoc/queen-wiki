====== Монтаж системы управления ======


Используйте наш обучающий курс **[video tutorials](video_tutorials_eng)** для электромонтажа системы управления.  


Если у вас установлен SketchUp, то можете исследовать нашу модель системы управления в 3D: **[queen control system sketchup model](https://1drv.ms/u/s!Am_hkdn5bouS1G9334yBP5ogC4-f)**.  


===== Рекомендуемый список дополнительных закупок для монтажа системы управления =====


После получения **[queen pack](queen_pack_eng)**, мы рекомендуем приобрести следующие позиции:  

| **item**            | **photo**                            | **pcs** | ** comments, requirements, models** |
| лист фанеры или текстолита    | {{:photo:plywood-1.jpg?direct&200|}} | 1 | необходим лист фанеры или текстолита габаритами не менее [630mm x 750mm] для монтажа на нем компонентов |
| электрический щиток | {{ :photo:electrobox-1.jpg?direct&200 |}} | 1 | опционально докупите электрический щиток для сокрытия компонентов и проводки |
| саморезы | {{ :photo:screw_1.jpg?50 |}} | много | тип зависит от того, что выбрано в качестве основания, текстолит или фанера |
| мультиметр | {{ :photo:multimeter_1.jpg?100 |}} | 1 | используйте мультиметр для проверки соединений в самом конце |

===== Подключение различных устройств =====

Используйте следующие типовые схемы для подключения устройств к QUEENBOARD:  


| **имя подключения** | **фото**            | **описание** |
| 12VDC-RELAY-CONTROL | {{ :layout:12vdc-relay-control.jpg?500 |}} | схема подключения 12V устройств к реле |
| 5VDC-RELAY-CONTROL | {{ :layout:5vdc-relay-control.jpg?500 |}} | схема подключения 5V устройств к реле |
| VAC-RELAY-CONTROL | {{ :layout:vac-relay-control.jpg?500 |}} | схема подключения 110/220V устройств к реле |

| 12VDC-PWM-CONTROL | {{ :layout:12vdc-pwm-control.jpg?500 |}} | схема подключения 12V устройств к ШИМ транзисторам для плавного управления |
| 5VDC-PWM-CONTROL | {{ :layout:5vdc-pwm-control.jpg?500 |}} | схема подключения 5V устройств к ШИМ транзисторам для плавного управления |

| DIN-PASSIVE | {{ :layout:din-passive.jpg?500 |}} | схема подключения датчиков с сухим контактом к дискретному входу |
| AIN-PASSIVE | {{ :layout:ain-passive.jpg?500 |}} | схема подключения датчиков с сухим контактом и резистивных элементов к аналоговому входу |
| DIN-ACTIVE | {{ :layout:din-active.jpg?500 |}} | схема подключения активных (требующих питания) датчиков к дискретному входу |

===== Проверка cоединений =====

После завершения электромонтажа используйте мультиметр для прозвона всех проводов до включения питания системы управления и после включения питания контрольного замера напряжений 24V, 12V and 5V на предмет работоспособности источников питания.
